# 仕事英語（社内IT向け）

瞬間英作アプリ（会話・瞬間英作）とは別に、**チケット・メール・定例**で使える定型と LLM 練習手順をまとめたメモです。

## 初めての人（ここから）

1. **[1週間スターター](getting-started.md)** — 毎日20分・フレーズと課題が決まっている
2. いちばん近い **シーン別の章** を1つ選ぶ（下記）
3. 送る前に **[送信前チェックリスト](templates.md#送信前チェックリスト5項目)** で確認

## 使い方（最短）

- まずは [フレーズ集](phrasebook.md) を見て、今日使えそうなものを5個選ぶ
- 「主語・期限・対象」だけ入れ替えて10回口に出す
- 週に2〜3回、[テンプレ](templates.md) で英文を書いて [LLMに添削](llm-workflow.md) してもらう

## 毎日の目安（20分）

[llm-workflow.md](llm-workflow.md) と同じ。スターター中は [getting-started.md](getting-started.md) の表に従う。

- 7分：フレーズ置換（口頭×10）
- 7分：短文を書く → LLM添削
- 6分：英語ドキュメント1段落の要約（または復習）

アプリ本体はリポジトリ直下の [README.md](../../README.md)。
