# 1週間スターター（初めての人向け）

毎日 **約20分**。フレーズは [phrasebook.md](phrasebook.md) の番号を参照しています。

| 日 | 今日の5フレーズ（phrasebook） | やること（分） |
|----|------------------------------|----------------|
| Day 1 | §1: Could you please ~? / by (Fri/EOD)? / When you have a moment / Would you mind ~ing? / Could you share ~? | 7分 口頭置換×10 / 7分 依頼メール3文を書く→[LLM添削](llm-workflow.md#2-添削学習になる形) / 6分 休憩 or 復習 |
| Day 2 | §2: Just to confirm / Could you clarify / What do you mean by / Do you mean A or B / more details on | 同上。題材：曖昧な仕様を英語で確認する3文 |
| Day 3 | §3: looking into it / investigating / can reproduce / can't reproduce / keep you posted | 同上。題材：調査中の返信3文 |
| Day 4 | §6: by EOD / by end of day (JST) / ETA is / aiming to complete / Does (Tue 3pm JST) work | 同上。題材：期限とタイムゾーンを明示した依頼 |
| Day 5 | §9: Thanks for reaching out / FYI / details below / attached / Let me know if you have questions | 7+7分。週末に [templates.md](templates.md) のチケット型で下書き1本 |

| Day 7 | 今週で一番使えそうだった5個を自分で選ぶ | [送信前チェックリスト](templates.md#送信前チェックリスト5項目) で週の下書きを見直す |

## 口頭置換のやり方（毎日共通）

1. 選んだフレーズ1つを読む  
2. 次だけ入れ替えて声に出す（10回）  
   - **主語**: we / I / they  
   - **期限**: by EOD / by Friday / by end of day (JST)  
   - **対象**: logs / schema / access / the API  

例（Day 1）:

- Could you please **share the error logs**?  
- Could you **share the schema** by **Friday**?  
- When you have a moment, could you **confirm the tenant ID**?
